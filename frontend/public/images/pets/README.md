# 宠物图片资源说明

本目录用于存放宠物展示图片。

## 图片尺寸建议
- 卡片图：400x400px
- 详情图：800x600px
- 格式：JPG 或 PNG

## 命名规则
- pet_1_golden_retriever.jpg
- pet_2_samoyed.jpg
- pet_3_labrador.jpg
